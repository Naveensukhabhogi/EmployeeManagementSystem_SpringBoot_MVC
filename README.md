# Employee Management System

## 📌 Project Description
Employee Management System is a Spring Boot application that performs CRUD operations on employee data.

## 🚀 Technologies Used
- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- Maven

## ✨ Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## 🛠️ Tools Used
- Eclipse IDE
- Git
- GitHub
- MySQL Workbench

## ▶️ How to Run
1. Clone the repository.
2. Open the project in Eclipse.
3. Configure MySQL database.
4. Run the Spring Boot application.
5. Open the application in your browser.
